# DevHire Backend

A simple recruitment backend system built with **NestJS** and **MySQL** 

---

## **Tech Stack**

- **Backend Framework**: NestJS
- **Database**: MySQL
- **API Testing**: Postman

---

## **Features**

- **User Authentication**
  - Register new users 
  - Login with JWT token
- **Job Module**
  - Create, read, delete job postings

---

## **Getting Started**

### 1. Clone repository

git clone https://github.com/yourusername/devhire-backend.git

cd devhire-backend

## 2. Install dependencies

npm install

## 3. Configure MySQL

CREATE DATABASE devhire;

(Update database credentials in src/app.module.ts)

## 5. Start the backend
npm run start:dev


Backend will run at: http://localhost:3000
